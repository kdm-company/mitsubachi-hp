# 株式会社MITSUBACHI コーポレートサイト

東大阪の産業機械・工作機械専門商社「株式会社MITSUBACHI」のコーポレートサイト（静的HTML）。

## 構成（5ページ）

| ファイル | 内容 |
| --- | --- |
| `index.html` | トップ（ファーストビュー・出来る事ダイジェスト・新着note・CTA） |
| `services.html` | MITSUBACHIに出来る事（サービス6項目の詳細） |
| `company.html` | 会社概要・代表挨拶・関連会社エイエルアイ |
| `news.html` | 新着情報（note RSS自動掲載予定） |
| `contact.html` | お問い合わせ（フォーム・電話・LINE） |

## デザイン

- カラーパレットは**ドラフト版**（ハニーイエロー・朱・藍・グリーン）。片岡鶴太郎の色使いを参照しつつコーポレート向けに調整。**クライアントの色confirm後に確定**すること（`css/style.css` の `:root` 変数で一括変更可）
- 機械メーカーの写真は使わず、やわらかなSVGイラスト（ミツバチ・六角形・草花）で表現
- フォント：Zen Maru Gothic（見出し）＋Noto Sans JP（本文）
- レスポンシブ対応（860px / 560px ブレークポイント）
